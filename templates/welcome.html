This web application serves as a lightweight frontend interface

Serving as the final layer of a larger automation workflow built using n8n, this web interface showcases structured candidate insights generated through an n8n-powered evaluation process.

📄 PDF Resume Extraction: Automatically extracts and processes resume content.

🧠 Smart Structuring: Organizes key information such as experience, education, and skills.

⚖️ AI Matching: Compares each resume with the job description using Word2Vec and cosine similarity.

         {click here to see deployed word2vec model}

🧮 Scoring System: Calculates weighted scores for each section of the resume.

🗃️ Database Integration: Saves results for easy access and tracking.

🏆 Top-Candidate Sorting: Displays the most qualified candidates through a user-friendly interface.
